  2. Greg Hurrell 2004-12-07T09:35:51Z

    No, the "Show" menu has been in there since version 1.1 (version 1.0 only showed the standard icon, and there were no other options hence no need for a menu).

    I put the "Show" menu in there because it's consistent with the pattern that you see in Apple's official menu extras. In order to take it out (or even offer the option to take it out), I'd want to know that there was a reasonable demand for the change, seeing as it would be a departure from the Apple style.

    I encourage you to open this up as a feature request in the bugs database (see http://bugs.wincent.com/); that way other users can vote for the change and I can gauge demand for the feature.
