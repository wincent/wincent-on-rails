  2. Greg Hurrell 2004-08-12T18:27:23Z

    This message is produced during normal operation of the MenuCracker menu extra enabler. MenuCracker is an opensource project hosted over at Source Forge; here is a link if you are interested in learning more about it. The project description is "Building a small bundle for MacOS X 10.2 in order to support menu extras in the same way they were supported in MacOS X 10.1".

    It's necessary to use MenuCracker (or another third-party menu extra enabler, such as the one from Unsanity) in order to use WinSwitch and be able to drag it around the menu bar while holding down the command key.

    I chose to not use the Unsanity enabler, because it requires their APE system.
