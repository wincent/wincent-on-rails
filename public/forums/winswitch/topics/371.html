  2. Greg Hurrell 2008-06-08T08:28:54Z

    WinSwitch is a Mac OS X program so if you are talking about running it inside Parallels (ie. inside Windows itself) then no, you can't use it inside Parallels/Windows. But you can use it on Mac OS X outside of Parallels.

    At the moment WinSwitch is compatible with Tiger but not Leopard. It is open source, however, so if anyone wants to submit a patch for Leopard compatibility then I will be happy to include it.
