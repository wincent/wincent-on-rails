WinSwitch menu item disappears

  1. Bob 2006-01-02T12:13:55Z

    I am using WinSwitch 3.1 on Mac OS X 10.4.3. The WinSwitch icon disappears every 2-3 logins and I must run "Activate WinSwitch" to get WinSwitch to show up again. How can I get WinSwitch to stay where it belongs in the menu bar?

  2. Greg Hurrell 2006-01-02T22:42:43Z

    I'd need more information to make an informed diagnosis; so here I'll just list off some possibilities:

    It might be going away because you access the "Accounts" preference pane. When you do that Mac OS X can try to add the Apple one back into the menu bar.

    Or failing that you might be logging into account for which the "Activate WinSwitch" script has not been run.

    Or failing that you might have something else on your system which is preventing either WinSwitch or the "MenuCracker" helper (which allows third-party menu extras to appear in the menu bar); perhaps you have a conflict with an APE module?

    Anything printed to the console?
