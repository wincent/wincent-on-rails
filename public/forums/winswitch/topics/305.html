  2. Greg Hurrell 2004-08-16T04:26:01Z

    Quote: Every time I switch user, the root user reappears in the menu list. Isn't it supposed to stay hidden?

    

    The preference whether to display the root user is set per-user. So it is perfectly possible for you to have the root user hidden in one account, and when you switch to another account, the display of the root user be enabled.
    

    By default the root user is not shown so in order to see it you'd have to manually switch it on. Quote: One other question: is there any way to switch users without requiring a password?
    

    You'd have to ask Apple for that one. WinSwitch only tells the system to switch users, and the system then throws up the password dialog. As a result, WinSwitch has no control over what appears in the dialog, and whether it asks for a password.
