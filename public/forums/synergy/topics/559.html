  5. Greg Hurrell 2014-10-21T01:32:06Z

    In the download archive if you look at "Extras" -> "Sample Button Sets" -> "About the Sample Button Sets.pdf" there is a section in the doc on how to make button sets. The main thing is just making a set of images (and they can have transparent backgrounds so you might be able to make some that look nice on the dark menu bar) which should be 22 pixels high and 72 DPI.

    As I said above, it would be nice to expand the format to include 2X "Retina" assets as well, which would look nicer on those screens, but that would require code changes.
