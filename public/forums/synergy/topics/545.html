  2. Greg Hurrell 2013-01-14T12:08:08Z

    In order to have Synergy start when iTunes start, you'd need another process that runs all the time watching to see when iTunes starts. So, we could make another process that did that, but if we're going to have an always-running background process we may as well just run Synergy. There are options to make it go out of sight when iTunes isn't active.

    Sorry about the memory usage. Synergy has had that memory footprint for close to a decade now, but there are no known leaks (ie. memory usage is stable over time), so it is unlikely that there'll be any significant changes to that footprint moving forward.
