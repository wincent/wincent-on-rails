Menu controls do not appear

  1. johnengels 2004-12-23T05:02:39Z

    I've installed Synergy on a G4 iBook using OS X 10.3.7, but the controls do not mount in the menu bar. (I've installed Unsanity's Menu Extra Enabler, and other menus appear as they should.) I tried it on my G4 iMac, also running 10.3.7, and it works fine. What's up?

  2. Greg Hurrell 2004-12-24T17:35:04Z

    The most common causes:

    1. Synergy is not running. You must start it by clicking the "Start" button in the Synergy pane of the System Preferences application.

    2. It is running, but your preferences instruct it to not show the menu bar control buttons. Go to the "Menu Bar" tab of the preferences and make sure you've got the appropriate options selected.

    3. It is running, but you have no room in your Menu Bar. Mac OS X will automatically hide the left-most menu extras so that your application menus always display. If you make more room, the left-most menu extras will reappear automatically. You can confirm this is the case by trying Synergy's hot keys. If the hot keys work but you don't see the menu buttons, then you know it's running but that Mac OS X has it hidden.
