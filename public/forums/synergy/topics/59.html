    Works for me here. Synergy doesn't use any hard-coded paths. It merely tells the system to launch "iTunes.app", and lets the system decide where that might be.

    If you've messed with your file system hierarchy then it's possible that Mac OS X LaunchServices doesn't know where iTunes.app is, in which case there's no way that Synergy could know either, short of scanning all mounted volumes.
