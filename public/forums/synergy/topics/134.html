  2. Greg Hurrell 2005-12-06T07:50:31Z

    Quote: Currently there's an option in Synergy to pre-process ID3 tags before searching amazon. Does this also work when playing from a shared iTunes library?

    Yes, ID3 tags will be pre-processed before searching Amazon regardless of the music source (shared playlist, CD, local files etc).

    Quote: Is there a way to turn amazon cover searching off and only have ID3 tag covers?

    There are some relevant feature requests in the database:

    http://wincent.com/a/support/bugs/show_bug.cgi?id=87

    http://wincent.com/a/support/bugs/show_bug.cgi?id=233

    http://wincent.com/a/support/bugs/show_bug.cgi?id=266

    Quote: Also perhaps only show the track/artist info and no "?" mark when there are no covers found?

    There's no concrete option in the UI for this. But you could hack this in yourself by substituting the corresponding graphic in the Synergy bundle:

    Code:Synergy.prefPane/Contents/Resources/Synergy.app/Contents/Resources/NoCoverArt.png

    You could replace that file with a blank (transparent) image of the same dimensions and name.

  4. Greg Hurrell 2005-12-07T02:38:24Z

    That's correct. If there's a cover in the ID3 tag then Synergy won't try to download BUT in the case of shared playlists iTunes doesn't pass album covers out to applications that request it via Apple Events and therefore it doesn't supply Synergy with the cover in the tag. This is a limitation of iTunes and it's what this bug report linked above is about; some kind of workaround will need to be found, or Apple will need to modify the behaviour of iTunes:

    http://wincent.com/a/support/bugs/show_bug.cgi?id=266

    This is why you are seeing a delay between playing and the cover showing up in those cases; Synergy is actually having to search for it and download it.

    As for the communication rate slider, yes, it's normal for it to be ghosted. If you are running a recent version of iTunes then Synergy doesn't need to poll iTunes and so the slider is irrelevant (ie. iTunes notifies Synergy automatically). On older versions of iTunes the slider is necessary because Synergy has to repeatedly poll iTunes.
