  2. Greg Hurrell 2004-11-23T10:18:13Z

    Can you post a screenshot of what the Floater looks like (the default key combo for capturing the screen is Command-Shift-3)?

    First thing I would do, though, is run the installer and click "Uninstall"; that will bring up a list of files and you'll be able to see if you've got two copies of Synergy installed (a common cause of login-glitches).

    If you have two installed, remove both, and install again.

    To get the login thing working again, uncheck the "Launch at login" setting in the prefs, click "Apply", then recheck it and click "Apply".
