  5. Greg Hurrell 2007-09-28T09:07:09Z

    This is going to be a post-Leopard thing seeing as I will be using Leopard as an excuse to start introducing post-Jaguar features (Synergy was released when Jaguar was the latest and greatest and I've maintained compatibility with Jaguar ever since then).

    So I'll continue to maintain the Jaguar branch with bug fixes for users still running the now-ancient version of the OS, but feature development will occur in new versions which rely on newer releases of Mac OS X.

    To implement this feature I'll be using a Core Data backing store, which is a Tiger-only thing. This and other stuff will start to trickle out when I start shipping the new builds, which will happen once Leopard goes public (next month, supposedly).
