Change in 1.2

  1. daemon 2004-08-28T05:32:59Z

    I have noticed that in Synergy 1.2 when you stream previews from the iTunes Music Store, the play/pause button in the menu bar disappears. I thought at first that this was because of a missing button in the button set, but it actually is because Synergy doesn't appear to have a handler for streaming previews for the play/pause button. But what is different from version 1.1 is that the button goes blank; in 1.1 it would show a button instead of making it disappear. Not sure if this change was desired or not, but I think that it isn't as visually appealing to have a missing button (especially in a button set like tabbed with the spacing set to 0). Thought it was worth noting.

    - daemon

  2. Greg Hurrell 2004-08-29T08:31:55Z

    Thanks for bringing this bug to my attention. I didn't know about it.

    Found and fixed. You are right that this started with 1.2.
