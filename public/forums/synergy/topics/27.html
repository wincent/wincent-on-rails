  2. Greg Hurrell 2004-06-29T14:05:25Z

    This is a historical legacy that stems from the fact that Synergy had artwork support before Apple ever released iTunes 4 (the first version which allowed you to embed artwork).

    So all that caching thing was Synergy's way of dealing with it before iTunes 4 came along. When iTunes 4 did arrive Synergy added the ability to take art from iTunes, but it continued using the cache because that was the "engine" of the whole cover art support.

    Synergy Advance doesn't have to work with this historical baggage, because it's a completely new product. You'll see some nifty features related to album art when Synergy Advance gets released...
