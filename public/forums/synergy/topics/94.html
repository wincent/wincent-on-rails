Album Cover Retrieval Suggestion

  1. import5byfive 2005-04-14T17:52:10Z

    For me, Synergy often retrieves incorrect album cover art. There should be a way to "notify" Synergy that it has retrieved the wrong image and have it look for another image. It should delete the incorrect album cover and perhaps keep an id tag of some sort of the wrong cover so that is doesn't download it again. Currently, if it downloads the wrong art, you have to do all the work yourself...find the correct image, download it, goto the album covers folder and replace wrong image. Synergy should be smarter in this respect, maybe even provide you with choices before it downloads the art if it comes across similarly named titles, like iTunes does when looking up album info from CDDB.

  2. Greg Hurrell 2005-04-15T15:20:45Z

    I agree it should be smarter. Note that you don't have to replace the wrong image though. An easier process is this:

    1. Find correct art 2. Drag into iTunes 3. Select "Show album covers folder" from Synergy Global Menu 4. Delete art

    In step 3 Synergy will select the art for the currently playing song, so steps 3 and 4 should be very quick. Synergy will use the correct art (from iTunes) the next time it plays the song.

    I have some ideas for making this easier on the user, and Synergy Advance will have a much better cover management system.
