  2. Greg Hurrell 2005-02-02T18:16:14Z

    Most likely the floater is popping up because iTunes is sending notifications with new track information. Perhaps the radio station you are listening to is getting a little bit too enthusiastic with the updates it embeds in the stream.

    You can monitor this yourself by using a notification monitoring program. You could probably find one on Version Tracker if you looked. I know that at least one exists, but I don't remember the name of it right now.
