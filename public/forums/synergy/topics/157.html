  2. Greg Hurrell 2006-05-03T06:10:23Z

    (1) Floater on the desktop

    This was implemented in version 1.3 (September 2004):

    Quote: [Feature] New "power user" feature (accessible from command-line), display floater on desktop. To activate, enter "defaults write org.wincent.Synergy desktopFloater -bool true" in the Terminal and restart Synergy. To deactivate, enter "defaults write org.wincent.Synergy desktopFloater -bool false".

    Due to lack of space in the preferences user interface I made this a command-line only customization. It's not surprising that you didn't know about it.

    (2) Always on with auto-hide if iTunes paused/stopped

    Thanks for the feature request. I've made an entry for it in the feature-requests database:

    http://wincent.com/a/support/bugs/show_bug.cgi?id=416

    (3) Bigger album art display

    There is already a feature request that's similar to this:

    http://wincent.com/a/support/bugs/show_bug.cgi?id=47

    But I've also made a new one that specifically describes what you're asking for:

    http://wincent.com/a/support/bugs/show_bug.cgi?id=417
