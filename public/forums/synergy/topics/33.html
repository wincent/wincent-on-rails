  3. Greg Hurrell 2004-08-12T18:59:29Z

    There are two places where you have to install the button sets, and one catch to be aware of. First, the places:

    1. Synergy.prefPane/Contents/Resouces/Synergy.app/Contents/Resources/buttons/ (for use by the Synergy.app which you see in the menu bar)

    2. Synergy.prefPane/Contents/Resouces/buttons/ (for use in the preference pane)

    Now, the catch:

    Synergy doesn't rescan the contents of those internal directories to see when new sets are added. This means that you can't actually add a new set; rather you must replace an old one. In other words, you need to pick a set you don't like, and overwrite the images from that set with your new images.

    In Synergy Advance you'll be able to add/delete/download sets in a number of easy ways. So far I have plans to be able to do all this within the preferences, or with drag and drop, or by double-clicking sets in the Finder.
