    It has worked fine, but stopped.

  2. Greg Hurrell 2004-02-25T20:12:25Z

    The most common cause of this type of problem is either:

    (1) You have more than one copy of Synergy installed

    or

    (2) Your loginwindow.plist has a "stale" entry in it pointing at an old copy of Synergy

    The solution for both cases:

    (1) Make sure you have the latest version (1.0a5 at the time of writing)

    (2) Run the installer and press the "Uninstall" button to get a list of all Synergy files on your computer. Perhaps there are two copies of the Synergy.prefPane in there?

    (3) In either case, remove the old copy/copies just to be sure, and reinstall.

    (4) In the preferences, uncheck the "Launch Synergy at login" checkbox and click "Apply" (this clears out any stale entry in the loginwindow.plist.

    (5) Re-check the checkbox and click "Apply".

    Voila. Everything should work. It's unfortunate that some users hit this kind of problem when upgrading.

    I have an update to Install (1.1) waiting in the wings that wil remedy this issue entirely by making sure there are no stale entries in the loginwindow.plist file every time you install a new version of Synergy.
