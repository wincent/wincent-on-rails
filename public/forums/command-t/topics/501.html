  2. Greg Hurrell Created 2012-01-11T15:23:42Z, edited 2012-02-15T06:24:34Z

    The Matcher constant that it is complaining about not finding is defined in Command-T's C extension.

    I suggest rebuilding Command-T (cd into ~/.vim/bundle/command-t/ruby/command-t and run ruby extconf.rb && make clean && make).
