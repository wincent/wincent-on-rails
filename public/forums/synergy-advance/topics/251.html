  11. Greg Hurrell 2004-12-22T05:24:16Z

    Hehe. I'm going crazy writing it too!

    Since this thread started I've solidified my plans a bit:

    (1) I've started to refer to the original Synergy as "Synergy Classic" on the website to avoid confusion with Synergy Advance.

    (2) Synergy Advance will support some Tiger-only features, but it will run on Panther also. The Tiger-only functionality will be disabled when running on Panther.

    (3) Synergy Classic will continue to run on Jaguar and Panther. At this stage, it currently runs on Tiger as well, and I hope that continues to be true when Apple releases the new OS.

    (4) Given the Tiger-only stuff in Synergy Advance, I've decided to coincide the release of Synergy Advance with the debut of Tiger. Apple hasn't yet announced a firm date for Tiger; we only know it's to be in the first half of 2005. I'll be doing my best to get Synergy Advance to the public at the same time.
