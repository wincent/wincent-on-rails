  2. Greg Hurrell 2005-07-31T11:30:03Z

    Not yet, but you will be able to do this in the future. You can use Synergy to download the art, and if you import it into iTunes then Synergy Advance will use it.

    The art handling will be completely different in Synergy Advance, supporting multiple sources (not just Amazon US) and providing a better UI for managing covers as well as doing things like importing back into iTunes.

    I don't have an exact ETA, but it will be pre-1.0. The next release will add the menu bar module, shortly after that you'll see control buttons, floater, and then cover art. At that point Synergy Advance will do everything that Synergy can do and considerably more, and then the real fun begins; I have quite a few more modules lined up after that.
