  2. Greg Hurrell 2006-06-11T10:40:43Z

    Until the Synergy Advance Floater plug-in is released, you'll have to use Growl to display notifications.

    The notifications can be customized in Synergy Advance's Growl preference pane, and there is a link to the growl site in the pane (http://growl.info/) if you need to download and install Growl.

    And in the hot keys pane you'll see hot keys for bringing up the Growl notifications. Let me know if you need any further help.

    To be notified when the Floater plug-in comes out, add yourself to the Synergy Advance announcements list:

    http://lists.wincent.com/

    You can also add yourself as a CC to this feature request:

    http://wincent.com/a/support/bugs/show_bug.cgi?id=432
