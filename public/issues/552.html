Feature request #552: REQUEST: Auto-play on launch (user-selectable song/playlist)

Kind feature request
Product Synergy
When 2007-03-19T06:21:59Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

This feature request is from a user email and really comes in two parts:

"a feature that will cause iTunes to go into play mode upon boot up? And play a specific song, play list or radio stream?"

My reply:

"[Y]ou can set Synergy to auto-start at login time and you can also set it to launch iTunes on demand. The missing piece of the puzzle would be to allow the user to turn on auto-play, and then the next logical step is the one you suggest (ability to define a specific song, playlist etc to be played)."
