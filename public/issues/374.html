Feature request #374: REQUEST: Show playlist name in Floater

Kind feature request
Product Synergy
When Created 2006-03-01T08:07:16Z, updated 2007-03-07T04:33:02Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

"Can you please add the ability to display the name of the Playlist selected that the music is coming from, or conversely, if it is being played from the Lib itself?

I have really gotten used to solely relying on Synergy, and go less and less to iTunes itself, but I'd like the floater to also display what Playlist the music is coming from. I have many "main" playlists I regularly use, as well as 2600 artist Playlists, and it's perfectly legal and desirable for a tune to be in more than one playlist.

I know it's just one more optional piece of data to clutter up the floater, but I believe it is very useful information, particularly if you have a large lib, and you want quick verification of where the music is coming from in the event it is a total mismatch, and should be removed. I would put the Playlist name on the bottom most line, and I would think you would also want some kind of Playlist icon at the beginning of the line so there can be no confusion as to what the data is."

Thanks for submitting the feature request. I think the best way to address the question of *where* the information should appear in the Floater is to provide the user with a way of customizing the layout; that's one of the things I'm working on.
