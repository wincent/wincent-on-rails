Bug #241: Multiple floaters on track change

Kind bug
Product Synergy
When Created 2005-05-17T11:14:01Z, updated 2005-07-13T20:34:44Z
Status closed
Reporter Matthew C Brooks
Tags no tags

Description

I get *three* floater windows when a track changes in iTunes. First a "classic" floater (grayish, transparent), and then two Growl floaters. The first Growl window says: Playing: [Title] [Album] [Artist], with the appropriate strings for the track playing. It has the default "artwork" of a CD with blue notes over it. The displayed items in this floater match my preferences as set in Synergy. The third floater is another Growl window that displays as: Now Playing Track: [Title] ([Rating]) Album: [Album] Artist: [Artist] This floater displays album artwork.

I'd like to somehow pare this down to ONE floater when a track changes. Preferably one that uses my preferences.

Help?

Comments

  1. Greg Hurrell 2005-05-17T11:26:46Z

    To change the notifications displayed by Growl:

    1. Go to:

    /Applications/System Preferences -> Growl preference pane

    2. Adjust settings to match your preferences.

    To turn the Synergy Floater on or off, go to the Synergy preferences and turn the Floater on or off:

    http://wincent.com/a/products/synergy-classic/preferences/general/

  2. Greg Hurrell 2005-07-13T20:34:44Z

    For reference, I've just added a new preference pane to Synergy Advance to provide more control over how/when it sends Growl notifications, and what gets sent. You'll be able to get your hands on it when the 0.2 public preview release comes out.
