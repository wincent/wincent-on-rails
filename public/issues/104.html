Feature request #104: Show grouping in floater

Kind feature request
Product Synergy Advance
When Created 2005-01-23T07:21:40Z, updated 2010-11-21T14:08:25Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

"I do have one request. I'd appreciate it if you could add a few more things as options for display in the floater. The one I really want is the Grouping tag -- I've begun using that as sort of a hyper-genre field, putting in one or more of about six "moods" I ascribe to a song (I use 2 letter codes for each mood). I then can use smart playlists to serve me up songs for the mood of the moment. It would be nice as I listen (not using these playlists) if I could check what Grouping codes I'd assigned."

To be honest, I had never really understood what the grouping field was for, but this is a good idea. Thanks for contributing the suggestion.
