Comments

  1. Greg Hurrell 2009-10-13T16:17:08Z

    There are two ways of changing button sets:

    • Selecting a new set from the already installed sets: In the preferences, choose a new set from the pop-up menu and click "Apply" (I presume you already know how to do this)
    • Downloading and installing a new set: After downloading, extract the archive by double-clicking on it. Install the new set by clicking on the extracted bundle (it should have a a .synergyButtons extension); this copies the set to the appropriate place in your ~/Library/Application Support folder and switches to it.

    If these aren't working for you, perhaps you have the "Choose random button style" option turned on (look in the "Advanced" tab of the preferences for that).
