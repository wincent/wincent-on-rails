Kind feature request
Product Synergy
When Created 2004-11-24T07:38:31Z, updated 2005-03-04T03:45:09Z
Status closed
Reporter Si Buckmaster
Tags no tags

Description

Hello, I've been a long time registered user of Synergy and I am glad to see you continually enhancing it. However, recently I have disabled Synergy and started to use Quicksilver to monitor/notify/control iTunes because it allows me to use the Growl notification system (growl.info).

Would it be feasible for Growl support to be added as an option for Synergy? I'd love to use it again but I also love my system-wide notification system. I've been avidly following development of Growl and it is being adopted by more and more software makers all the time, from what I hear they provide an excellent SDK.

Thanks for your time (and I apologise if I have misunderstood the feature request procedure, but there didn't seem to be any differentiation between bugs and features on the website).
