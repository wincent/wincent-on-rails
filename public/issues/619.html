Feature request #619: REQUEST: Forward/back album (hot keys, menu item or control button modifier)

Kind feature request
Product Synergy
When 2007-11-29T14:01:58Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

I thought there might already be a request for this in the database but I can't find it so opening a new one. Just got this email from a user:

--

- I'd like a menubar button, or a modifier (option key) with the current buttons, to go to next/previous album (I have set the shuffle preference in iTunes to "album"). There is no AppleScript command to do this directly (although option-click in iTunes's buttons work), but a script like this should work (but it does not worry about infinite loops or end of playlist...):

tell application "iTunes" set previousAlbum to album of current track repeat until false next track if album of current track is not previousAlbum then exit repeat end repeat end tell
