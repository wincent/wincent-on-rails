Feature request #257: REQUEST: Editing phrases filtered out when searching Amazon.com for album covers

Kind feature request
Product Synergy
When Created 2005-06-12T20:09:35Z, updated 2006-07-05T02:06:00Z
Status open
Reporter Fabio Vitali
Tags no tags

Description

I would really appreciate being able to maintain my own personal list of phrases you should be filtering out when searching Amazon.com.

For instance, I use "OST" instead of "Soundtrack", "disco 1" and so on instead of "disc 1", and I always prepend the year before the album title (so that it is sorted chronologically in the album panel in iTunes).

If I was able to say that, for instance, "2003" is not part of the title, the Amazon search would actually work for me.

Thanks

Fabio
