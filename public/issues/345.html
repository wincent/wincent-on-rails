Feature request #345: REQUEST: Customizable Floater shape

Kind feature request
Product Synergy
When Created 2005-12-20T18:58:55Z, updated 2006-07-05T02:07:07Z
Status open
Reporter horkance boofblatt
Tags no tags

Description

The floater window is great. Initially, I was thinking that scrolling text in the menu bar would be great, but now I think that if there were more flexibility for the floater, I wouldn't care about the scrolling text.

It would be great to fully configure the floater:

 1. Background box color (black, green, red, whatever)
 2. Background box shape (square corners, rounded, etc)
 3. Font face, color and size for the different text elements
 4. Location of text elements
 5. Boundary padding between text and border
 6. Remaining time preference

You probably have so much free time on your hands that you'll get to this request tomorrow and have it in production by the end of the week.. ;-)

Even without these requested features, it's a great product.
