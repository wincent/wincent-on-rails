Bug #658: Show play/pause bezel on notification from iTunes

Kind bug
Product Synergy
When 2008-03-07T13:38:50Z
Status open
Reporter Nik Gervae
Tags no tags

Description

I just got a new MacBook Pro, with dedicated iTunes control buttons on the F-keys. When I use my Synergy hotkey to play/pause, it shows the little bezel. When I use the F-key play/pause one, Synergy is bypassed. If iTunes sends out any kind of notification, or if you can discover that some way, it would be nice if Synergy showed it no matter what triggered the change (F-key, Quicksilver, or random AppleScript).
