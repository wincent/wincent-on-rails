Feature request #517: REQUEST: Submit tracks played on iPod to last.fm

Kind feature request
Product Synergy
When Created 2006-12-04T20:21:33Z, updated 2009-01-05T16:24:24Z
Status open
Reporter Elaine Wilson
Tags no tags

Description

One of the reasons I will continue to use iScrobbler rather than Synergy's last.fm feature is the ability to upload ALL my recently played tracks to last.fm, not just those played locally. The relevant preference in iScrobbler reads: "Enable iPod Submissions Using Playlist: " ... I use "Recently Played," the default smart playlist of that name. This way the podcasts and other tracks I listen to away from home still make it to my last.fm profile when I get home and sync my iPod.
