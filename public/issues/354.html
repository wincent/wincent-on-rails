Bug #354: Display/Hide of Synergy Icon in the menu

Kind bug
Product Synergy
When Created 2006-01-24T11:55:28Z, updated 2006-07-05T02:07:14Z
Status open
Reporter Michael Nordmeyer
Tags no tags

Description

I disabled the menu buttons to safe some space which I configured before to only show if iTunes is running. Now I want to display the standard icon (the musical note, don't know how this is called in English) to get access to the playlist. settings etc. but only if iTunes is running. Currently the icon is always displayed and the spproriate setting is greyed out in the preference panel.

So I think this would be a feature request.

Comments

  1. Greg Hurrell 2006-01-24T13:00:18Z

    Thanks for the feature request. It's true that you can't currently set the Global Menu to show "only when iTunes is running". You can set it up to display all the time. And you can set it up to do the opposite (only when iTunes is NOT running) but only when you have the buttons set to "Display controls only when iTunes is running".

    So a new option would need to be added, "Only when iTunes is not running", underneath "Show Global Menu as a separate menu item". Or two radio buttons "Only when iTunes is not running"/"Only when iTunes is running". The two radio buttons are probably a better idea because I am sure that as soon as I implement the one, people will ask for the other!
