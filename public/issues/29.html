Comments

  1. Greg Hurrell 2004-11-18T11:40:06Z

    Are you on a small screen? I'll bet that what's happening is that the left-most menu extras (in this case, Synergy) are being temporarily hidden because Mail.app has a lot of menus. This will happen in any application which occupies a lot of menu space.

    If this is what's happening, it's not a bug, but is standard behaviour for Mac OS X. You can test it out by running apps which use a lot of menu bar space (on the left), and putting lots of menu extras on the right. You'll see that they get hidden by the operating system and reappear when you switch to an app which doesn't obscure them. This applies not only to Synergy but to anything that runs on the right side of the menu bar.

    You'll notice that Synergy is still running while it is obscured, because you can still use its hot keys and see the floater.

    Let me know if this is what's happening and I'll close this bug.
