Feature request #12: REQUEST: Floater integration with screensaver

Kind feature request
Product Synergy
When Created 2004-11-10T00:37:55Z, updated 2006-07-05T02:02:53Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email: "the floater could link to screen saver showing a larger view of the album cover floating around or zooming in and out. Mostly when I play music the computer sets idle, and the screen saver comes on"

I doubt that it's possible for the floater to display itself on top of a running screensaver, but the obvious alternative is to include an optional, separate screensaver that shows album art.
