Bug #372: Add About when user does not have Synergy Advance in Dock

When 2006-02-26T01:03:25Z

Description

It would be nice to know the version of Synergy Advance when the application is set to not show in Dock. This could be as easy as another option in the Preferences.

Comments

  1. Greg Hurrell 2006-02-26T06:35:01Z

    Thanks for the submission, Travis. There are currently five ways of finding out the version of Synergy Advance:

    1. Do a "Get Info" on it in the Finder.

    2. Look at the "splash" window on launch (the splash window can be turned on and off in the Advanced tab of the prefrences).

    3. Choose "About Synergy Advance" from the application menu (if Synergy Advance is frontmost).

    4. Choose "About..." from the Global Menu (requires you to hold down the Option key to see the alternate menu options in the menu).

    5. (For nerds only), run the what(1) command-line tool on the executable like this:

    $ cd /Applications/Synergy\ Advance.app/Contents/MacOS/ $ what Synergy\ Advance

    Options 2, 3, 4 and 5 also should the build-number. Option 1 only shows the marketing version number.

    Given that there are so many ways to know the version I'm going to mark this one as INVALID/CLOSED because I suspect you didn't know about them. If you have a suggestion for another, different way of showing the version then feel free to re-open this issue and add a comment.
