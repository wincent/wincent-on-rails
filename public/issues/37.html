  2. Greg Hurrell 2004-11-24T23:42:31Z

    Just to put this advice on the record, there are a number of steps you should undertake when you encounter a problem like this to find out whether the problem lies in Synergy or outside of it (in it's environment). Once we know what's causing the problem we can then either fix Synergy to cope with the environment, or fix the environment!

    1. Uninstall using the installer, then reinstall.

    2. Quit Synergy, drag your preferences file out onto the Desktop, then restart Synergy.

    3. Try to reproduce the problem from another user account on the same computer; if you don't have another account, make one.

    If "1" fixes it then, er, it's fixed.

    If "2" fixes it then you know you had a corrupt preferences file.

    If "3" fixes it then you know there's something in your user account which is triggering the problem (other software you're running? conflicts? etc); this we can investigate further until we know what's triggering the problem.

    If none of these fix it then we are talking about a bug in Synergy, or a conflict triggered by your hardware, peripherals, OS version etc.
