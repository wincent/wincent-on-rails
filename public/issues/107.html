Feature request #107: REQUEST: Quicktime Player control

Kind feature request
Product Synergy Advance
When Created 2005-01-27T10:02:18Z, updated 2006-07-05T02:04:09Z
Status open
Reporter Jani G
Tags no tags

Description

This feature would allow quicktime to be controlled like iTunes if Quicktime is open when iTunes is not. For example if you have a audio or video file playing in Quicktime (and iTunes is not running) while Synergy is running, using the same key combos would produce same effect to Quicktime as they usually do to iTunes. A user could then for example use Command + Right Arrow to forward through a Quicktime file, just like iTunes.
