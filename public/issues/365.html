Feature request #365: Support for showing/settting half-star ratings in iTunes

Kind feature request
Product Synergy
When Created 2006-02-17T00:47:56Z, updated 2010-06-18T11:30:46Z
Status open
Reporter Aaron Prouty
Tags no tags

Description

Although not directly implemented, iTunes has half-star rating capability. Currently Synergy's floater window will not display ANY rating for half-star songs. It would be great to see this addressed in the next release.

Other features that would be great additions to the program would be: 1. Hotkeys to rate songs as half-stars, or increase/decrease rating by half-stars 2. "Empty" hotkey slots for the user to attach custom scripts to

    Thanks for submitting the request/report, Aaron.

    There are really three separate issues here so I've split this into three different issues.

    1. The total non-display of any rating which is not a whole-star multiple (a bug). I hadn't noticed that myself but will check it out right now. I've opened a separate report for this one (bug #366).

    2. The feature request for half-star hot keys. We'll leave this issue that you've opened for that (#365). For the record, Synergy Advance already supports both the setting and display of half-star ratings, and in fact you can make it set stars using an even finer interval if you want (although half-star is the finest resolution that you'll be able to see in the display). The currently released version has the support for setting such ratings and partial support for displaying them (ie. in hot keys bezels); the soon-to-be-released version has enhanced support for display of such ratings (ie. in Growl notifications and elsewhere).

    3. The feature request for empty hot key slots. I've opened a separate report for that (feature request #367)

  5. Greg Hurrell 2007-05-14T09:35:48Z

    (In reply to comment #4)

    Well, Synergy Advance displays half-star ratings fine, but I can't figure out any way to use it to increase/decrease ratings by 1/2 star. The way I use it now (using a hacked version of the quicksilver "increase rating" script) is to set the rating using 1-5 stars (cmd-shift-1 through 5) and then increase/decrease by 1/2 star from there.

    If you're using Synergy Advance 0.4 then you can assign hot keys for "increase" and "decrease rating", and under the "Advanced" pane in the preferences set the "Rating change step" to 10 instead of the default 20. This allows you to move ratings up and down by half-star intervals via hot keys. For more info see:

    http://wincent.com/a/products/synergy-advance/help/preferences/advanced/#rating-step

    In Synergy Advance 0.5b that setting still works but the preferences UI has been (and still is being) rearranged so you'll have to make the setting using the terminal:

    defaults write com.wincent.SynergyAdvance RatingChangeIncrement -int 10

    This is only temporary; full access to all the settings via the UI will be restored once the preferences rework is all done.

    I'm also looking at other ways of allowing users to set half-star ratings (additional hot keys, and via the menu). See feature request #240.
