Feature request #373: REQUEST: Play-order should match menu-order of tracks selected from Global Menu

Kind feature request
Product Synergy Advance
When Created 2006-02-26T01:07:43Z, updated 2006-07-05T02:07:23Z
Status open
Reporter Travis Conway
Tags no tags

Description

When a user chooses a track from the Synergy Advance menu, Synergy stops after playing track. It would be best to offer a menu option to continue playing tracks in the order in which they were displayed in the menu chosen. For example, if I choose David Gray => All => Freedom then it should go to the next track under the All menu, but if I choose David Gray => A New Day at Midnight => Freedom, then it should go to the next track under A New Day at Midnight. This would be great for me to choose an artist and cycle through all their tracks without stopping, bring up iTunes, choose the tracks, and hit Play.
