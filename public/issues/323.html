Feature request #323: REQUEST: Hot Key to open the Synergy Advance Preferences

Kind feature request

Status open

Description

In the Synergy Advance portion of the Hot Keys preference pane it would be nice to be able to assign a hot key combination for opening the SA Preferences. I don't use the dock icon or menu bar icon (I still use the Synergy Classic menu bar) for Synergy Advance.
