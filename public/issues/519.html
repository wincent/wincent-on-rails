Bug #519: Non-translated strings in Synergy 3.1

Kind bug
Product Synergy
When Created 2006-12-06T07:59:57Z, updated 2007-01-15T03:09:01Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

'After downloading and installing Synergy 3.1 I noticed that several strings were not translated to german inside this new release, such as "Transfer cover to iTunes" (should be "Cover zu iTunes senden") inside the menu, or the whole Audioscrobbler part. The same appears to an error message that I encountered during the installation "There was a problem moving the item .... to the trash ....".'

I've already applied the mentioned change to the "Transfer cover to iTunes" string. I'm opening this issue so that if anyone sees other strings that need translation there is now an open issue in the database.
