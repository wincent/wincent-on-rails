  2. Greg Hurrell 2009-02-21T03:54:29Z

    1. Synergy asks iTunes for the art, so if nothing appears in the Floater it is because for some reason iTunes isn't handing it over. What versions of Synergy, iTunes and Mac OS X are you running? And does this happen for all tracks or just some of them? If the latter, what kind of tracks are failing to deliver art? Finally, have you tried any of the troubleshooting hints (such as ruling out preference file corruption and trying to reproduce from a different account) that are listed at http://wincent.com/s/bugs/?

    2. Not sure what your question is, so I'll just clarify that there are actually three places that art might be stored:

    • Inside the music files themselves.
    • Inside Synergy's "Album Covers" folder; this is for art that gets downloaded from the Internet when iTunes doesn't have or provide art.
    • Inside a cache folder, "Temporary Album Covers"; this folder contains copies of the art that iTunes hands over, because it is faster to read the art from a file on the disk than to request it from iTunes via Apple Events. This folder is automatically cleared out whenever Synergy quits so you don't have to worry about it filling up with cached artwork.

    3. See feature request #414, and also some additional notes on duplicate feature request #543.
