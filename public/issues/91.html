Feature request #91: REQUEST: Option to disable hot keys when iTunes is not running

Kind feature request
Product Synergy Advance
When Created 2005-01-05T06:46:31Z, updated 2005-05-03T09:39:35Z
Status closed
Reporter John Balestrieri
Tags no tags

Description

I'd like to have an option to turn off the hot keys when iTunes is not running. Sometimes I accidentally hit a synergy hot key, and if iTunes is not running, it launches -- causing me to force quit iTunes, etc. It would be simpler if there were the option to have hot keys active only when iTunes is running.
