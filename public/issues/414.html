Feature request #414: REQUEST: Auto-launch Synergy when iTunes launches

Kind feature request
Product Synergy
When Created 2006-04-27T11:49:45Z, updated 2007-02-09T08:59:09Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

"Can Synergy 2.7 be configured to automatically start when itunes 6.0.4 starts? How? If not, please add this feature as I have an older mac running 10.3.9. Thank you in advance. "

The short answer: no it can't be configured.

The long answer: the reason it can't be configured to do so is that in order to watch for iTunes launching something else has to already by running (to watch for the launches). So you'd need a separate process, let's call it "iTunes watcher" that would launch at login and watch for iTunes launches, then launch Synergy when appropriate.

If you're going to run an extra process that watches iTunes you may as well just let Synergy run at login and do the watching itself. There are preferences for hiding Synergy when iTunes is not running.

A side note:

This feature is already implemented in Synergy Advance, which includes a light-weight daemon called synergyd for precisely this purpose. I'm opening this feature request in the database to see if there's sufficient interest among users for adding this feature to Synergy as well.
