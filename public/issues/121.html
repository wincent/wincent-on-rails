Feature request #121: REQUEST: Option to toggle shadow on the floater

Kind feature request
Product Synergy
When Created 2005-02-24T14:40:51Z, updated 2006-07-05T02:04:17Z
Status open
Reporter Mario Estrada
Tags no tags

Description

The Bezel should not draw a shadow, none of the bezels that comes with the OS draw them... It could be an option on the preferences...

Comments

  1. Greg Hurrell 2005-02-24T21:06:05Z

    Thanks for submitting the request, Mario. And well spotted, too; you have a sharp eye!

    If you look carefully, you'll see that the feedback bezels which pop up (if you have them turned on in the preferences) when you use the hot keys to play/pause etc do not have shadows, just like the system bezels. I've done my best to make these bezels exactly match the ones in the OS.

    The floater, on the other hand, does have a shadow as you've pointed out. I've slightly reworded the summary so that it includes the word "option", and also says "floater" instead of "bezel". It will make sense that the floater should have a shadow in some cases, and not in others, especially once I add more options to control its appearance (in other words, in the future, you will should be able to make the floater look quite unlike a bezel, if you want)...
