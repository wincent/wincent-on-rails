Feature request #283: Auto-hide iTunes on launch

Kind feature request
Product Synergy Advance
When Created 2005-07-31T03:18:02Z, updated 2008-09-03T04:51:26Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

"If iTunes must launch so Synergy can work, could it remain supressed to the Dock without opening the application so that I have to minimize it to the Dock.

Sometimes when I'm working, I click on Synergy to start playing, but then I'm briefly interruptede by iTunes and I have to do something to minimize it."

I think this sounds like a reasonable request but for obvious reasons it would have to be: (1) optional; and (2) applicable only when Synergy Advance is used to launch iTunes (and not when iTunes is launched by double-clicking on it in the Finder, launching it from the Dock, or launching it from Spotlight).
