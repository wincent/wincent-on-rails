Feature request #550: REQUEST: Add chapter navigation hotkeys

Kind feature request
Product Synergy
When 2007-03-08T14:53:09Z
Status open
Reporter Hugo Schotman
Tags no tags

Description

Audiobooks and a lot of (so called 'enhanced') podcasts have chapters defined. It would be very beneficial if we could navigate through those with hotkeys. I have many podcasts with > 30 chapters that I frequently want to skip through. Right now I'd have to open iTunes to do that.

Thanks, keep up the good work!

Comments

  1. Greg Hurrell 2007-03-08T17:28:46Z

    Thanks a lot for submitting the feature request. I'll have to investigate and see if iTunes provides some kind of scripting access to its chapter navigation features.

    As a temporary substitute you might want to check out the public preview release of Synergy Advance, which allows you to set hot keys for skipping forward and back by a fixed amount of seconds (and it also allows you to set how may seconds that fixed amount should consist of).

    Also, you can seek within tracks by pressing and holding the "next/previous" hot keys (in both Synergy and Synergy Advance) and the "next/previous" menu bar control buttons (in Synergy). Admittedly not chapter skipping, but it's still an alternative.

    Thanks once again for filing the request!
