Description

1.9.2 seems to be doing fine right now, but locally (on Mac OS X under RVM) it's proving to be flakey and segfaulting.

As I've updated my local box to 1.9.3 to avoid the segfaults, I'd like to update the production system as well for consistency between development and production environments.
