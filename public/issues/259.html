Bug #259: Keystroke rating change results in up to 5 separate Growlings

Kind bug
Product Synergy Advance
When Created 2005-06-15T15:00:31Z, updated 2005-07-27T01:33:02Z
Status closed
Reporter William Styler
Tags no tags

Description

Hiya. This is pure polish, but I've noticed that when I use keystrokes to up a song's rating incrementally when using growl, it generates up to 5 different growls. When using bubbles or another stacking style, that's alright, but it's a bit of a pain with music video style or others that act similarly. Perhaps there could be a way to bring up a bezel outside of growl or something along those lines? Maybe even a new notification type in growl so that I could set it to a different style than track info. I'm not entirely sure what the best way to fix it would be (or if it even needs fixing), but I figured I'd put it out there.

Thanks again!
