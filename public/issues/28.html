Feature request #28: REQUEST: Control floater background color

Kind feature request
Product Synergy
When Created 2004-11-18T02:03:42Z, updated 2006-07-05T02:03:01Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email: "completely transparent background"

I'm filing this feature request as "Control floater colors and transparency" because in order to support a completely transparent background will probably have to support manipulation of the text color as well.

In Synergy Advance I'm using Apple's default color picker for this, which allows you to not only choose any color, but also set any transparency level between 0 and 100%.
