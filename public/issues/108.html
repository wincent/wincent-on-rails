Feature request #108: REQUEST: Integrated Audioscrobbler (last.fm) support

Kind feature request
Product Synergy Advance
When Created 2005-01-28T04:57:43Z, updated 2006-12-04T12:41:26Z
Status closed
Reporter bugzilla user 82
Tags no tags

Description

As I love the functionality of Synergy and having all of my iTunes toys centralized in one running application (I use Rating Bar and AudioScrobbler in addition to Synergy and QuickSilver), I would love to see you add built-in support for Audioscrobbler.com.

All it would need to do is simply upload my playlist to Audioscrobbler and allow me to access my statistics page.

While nothing in software development is trivial, this functionality makes complete sense for Synergy.
