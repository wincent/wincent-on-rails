Feature request #392: REQUEST: Hot keys for "Play" and "Stop"

Kind feature request
Product Synergy
When Created 2006-04-04T05:17:14Z, updated 2006-07-05T02:07:36Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

"Hi, I want to take this opportunity to point out something that has been bothering me for some time. I’ve noticed that there are no commands or Hot Keys for the Stop and Play commands. Pause/Play is not what I want. I know that iTunes will respond to those commands via AppleScript. I’m more than a little mystified by the fact that you would dedicate eight Hot Keys fields to ratings and stars. I mean who cares about ratings and stars anyway?... Let’s do something useful with those fields, please."

There are a lot of users who do care about ratings so I can't really get rid of any existing hot keys. This request really depends on feature request #294, which is about modifying the user interface so that more hot keys can be fit in.

For reference, Synergy Advance has hot keys for "Play", "Stop" and a bunch of other stuff, so you could user that in the meantime (free trial available) while waiting for this request to be fulfilled in Synergy.
