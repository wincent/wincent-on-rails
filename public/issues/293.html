Feature request #293: REQUEST: Hot key to programmatically "press" curly arrow button

Kind feature request
Product Synergy Advance
When Created 2005-09-08T19:44:17Z, updated 2006-07-05T02:06:27Z
Status open
Reporter froodiantherapy
Tags no tags

Description

I tend to use a hybrid of techniques to control iTunes, so the source selected in the iTunes source list at any given time is rarely the same source as the song currently being played. Despite this, I want to be able to use my Play/Pause hotkey for pause, never for stop. To accomplish this, I thought it would be nice to be able to set a hotkey to effectively click the curley arrow button in iTunes (without bringing it to the front).
