Bug #660: Album artwork only come up when synergy preferences are manipulated

Kind bug
Product Synergy
When Created 2008-03-10T16:39:04Z, updated 2008-03-12T06:25:32Z
Status closed
Reporter bugzilla user 535
Tags no tags

Description

AS I use this porduct, the album artwork does not automatically display after the songs switches. If i go into the synergy preferences and manipulate and option the artwork miraculusly appears. I paid for this because of its ability to display artwork and I can concerned that after paying for the license, my product no longer works

Comments

  1. Greg Hurrell 2008-03-12T06:24:44Z

    Are you using iTunes 7.6.1, by any chance? In that case, this is a known issue; see:

    http://wincent.com/a/support/bugs/show_bug.cgi?id=601

    Apple introduced a change in 7.6.1 which I am hoping they will back out in 7.6.2. But your bug report does shed some light on a possible workaround: basically it looks like iTunes is notifying that a new track starts playing, but when Synergy asks it for the artwork it doesn't have it ready yet; so the workaround may be to delay asking for the artwork by some small interval.
