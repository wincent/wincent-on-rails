Bug #2170: Synergy quits upon restart

Kind bug
Product Synergy
When Created 2014-04-30T21:22:27Z, updated 2014-06-16T03:46:41Z
Status closed
Reporter anonymous
Tags no tags

Description

Synergy quits after a restart or shut down. It will not open automatically upon login. Must be restarted from the System Preferences panel.

Running a MBP 13" 2.9GHz 8 GB RAM running 10.8.5 fully updated.

Comments

  1. Greg Hurrell 2014-05-01T14:05:11Z

    I'd try toggling the "start at login" feature on and off.

    If that doesn't work, I'd try manually checking the list of login items (System Preferences > Users & Groups > Login Items) and if necessary dragging the Synergy app to the list (depending on where you installed it, the app is probably at /Applications/Synergy Preferences.app/Contents/Resources/Synergy.app, and you can look inside the Synergy Preferences.app package by right-clicking on it and choosing "Show Package Contents").
