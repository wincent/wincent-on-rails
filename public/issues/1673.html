Feedback #1673: Confirmed working versions on Windows XP

Kind feedback
Product Command-T
When Created 2010-09-01T09:49:24Z, updated 2010-09-02T13:28:03Z
Status closed
Reporter anonymous
Tags no tags

Description

Tried a lot of combinations of Vim (7.2, 7.3, "Cream" version, Cygwin version), Ruby (official version from ruby-lang.org, various versions from rubyinstaller.org) and compilers (rubyinstaller.org DevKit, Cygwin, MinGW etc).

The only combination I could get to work was:

  • Official Vim 7.2 from www.vim.org: ftp://ftp.vim.org/pub/vim/pc/gvim72.exe
  • Ruby and DevKit from rubyinstaller.org, but not the latest versions; instead, from the http://rubyinstaller.org/downloads/archives page:
  • Ruby 1.8.7-p299: http://rubyforge.org/frs/download.php/71492/rubyinstaller-1.8.7-p299.exe
  • DevKit 3.4.5r3-20091110: http://rubyforge.org/frs/download.php/66888/devkit-3.4.5r3-20091110.7z

It may be that other combinations work if you use the incantations in the right order, but this was the only combo that worked for me on Windows XP.
