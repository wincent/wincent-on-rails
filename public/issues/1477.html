Feature request #1477: Change column order in issues listing

Kind feature request
Product wincent.com
When Created 2010-02-11T17:31:45Z, updated 2010-06-24T09:34:06Z
Status closed
Reporter Greg Hurrell
Tags no tags

Description

Current order is:

  • Public/private icon (administrator view only)
  • Kind
  • Number
  • Product
  • Summary
  • Status
  • Timestamp

Would probably be nicer if first two were:

  • Number
  • Summary

As though are the most important aspects of any ticket. Remaining columns could then be:

  • Status
  • Product
  • Kind
  • Timestamp

The order of these later columns is not so important.

The "public/private" column should probably be eliminated, with private issues just acquiring a padlock icon rolled into one of the other columns (the "status" column perhaps? or right-aligned in the "summary" column?)
