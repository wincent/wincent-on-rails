Bug #421: Can't "drill down" in "Genres" and "Artists" submenus

Kind bug
Product Synergy Advance
When Created 2006-05-06T04:26:12Z, updated 2007-05-15T12:22:28Z
Status closed
Reporter Greg Hurrell
Tags no tags

Description

As noted in the 0.4 beta release notes (http://lists.wincent.com/pipermail/synergy-advance-beta/2006-March/000000.html) there is a regression in version 0.4:

  • The "Genres" and "Artists" menu items only show albums; they do not allow you to "drill" down to the level of individual tracks. This is because much of the Global Menu code has been rewritten and there is

still a small amount of work to be done before full functionality is achieved.

Will be exploring this issue today and hopefully will be able to release a build that contains a fix shortly.
