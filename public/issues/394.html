Bug #394: Unwanted playlist switching on play/pause

Kind bug
Product Synergy Advance
When Created 2006-04-04T09:42:44Z, updated 2006-05-07T07:33:23Z
Status closed
Reporter bugzilla user 330
Tags no tags

Description

Hi,

I've written to you before about this. When I choose a song from a playlist via Synergy Advance, it goes to that song. Great. If I then choose a different song from a different playlist, it plays that song. Great.

But if I then hit "pause," it reverts to the previous song in the previous playlist when I hit "play" again. Most annoying. It should stay on the chosen playlist.

Butler also allows me to select specific songs from specific playlists, but without any problems. You might want to look at how they do it.
