Feature request #102: REQUEST: Hot key to cycle through Amazon search results

Kind feature request
Product Synergy Advance
When Created 2005-01-19T08:47:50Z, updated 2006-07-05T02:04:08Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

"unfortunately Synergy often does not find the right album cover for my CDs. Therefore I would welcome a hotkey so that I can tell Synergy easily that it is the wrong one. Perhaps it can look for another candidate at Amazon then next time."

The way Synergy currently works, it performs the search at Amazon and assumes that the number one search result is the correct one.

In Synergy Advance I intend to do what you suggest and add a hot key (or a mouse-based method) for cycling through the search results if the top one is not correct.
