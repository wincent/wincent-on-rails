Kind bug
Product Synergy
When Created 2008-01-18T15:06:02Z, updated 2008-09-14T17:02:14Z
Status closed
Reporter reppep
Tags no tags

Description

I took all the current Software Updates (including iTunes) today and rebooted. Synergy didn't show up in my menu bar, and my keyboard shortcuts don't work.

I reinstalled 3.2, and observed that I could Start Synergy from System Preferences, but not Stop it. I removed and added the startup item from Synergy's prefpane too. Further, I could kill it with "killall Synergy", but this had no visible effect (except that the next time there were no matches to kill). Since the upgrade, ATSServer is dying a lot, but other apps are coping.

This is 10.5.1 on a Mac Pro.

In addition, iTunes 7.6 (20) reset my Library to factory defaults (empty; stock playlists only)
