Bug #254: Rating does not display when not exactly a multiple of 20

Kind bug
Product Synergy Advance
When Created 2005-06-10T08:15:03Z, updated 2006-07-05T02:06:00Z
Status open
Reporter David Ameeti
Tags no tags

Description

If I change the rating increment for hot keys to 10 (1/2 a star) and then use one of the hot keys to change the rating to 4.5 stars, the transparent window works such that it shows 4.5 stars. BUT... when I move to another song then then come back, when it displays the normal "this song is playing" transparent window, the rating is NONE (i.e., no stars shown). If I then use the hot key to change the rating (bump up by 10 pts), it correctly moves to the next appropriate rating as though it did save the rating properly. If I change the rating to a multiple of 20 (i.e., a normal full star level), all works well. It sounds like it's only a display problem since the rating *was* set properly -- it just does not display the stars when you play the song later if it's not exactly full stars.
