Comments

  1. Greg Hurrell 2011-02-15T08:29:49Z

    Licenses are case-sensitive and tied to the email address used at the time of purchase, so make sure you are using the serial number and email address exactly as they appear on your license notice. I recommend using copy and paste for obvious reasons.

    If you're still having trouble send me an email (to win@wincent.com) and if necessary I can send you a screenshot showing what successful registration looks like.

    The other possibility is that you might be using a very old serial number in a recent version of Synergy. If that is the case see this page.
