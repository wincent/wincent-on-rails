  10. Greg Hurrell 2009-05-26T18:02:35Z

    Here's a little test I just did in the Safari snippet editor:

    <style type="text/css">
  pre span.line-number {
    counter-increment: line-number;
    text-align: right;
    float:left;
    padding-right: 1em;
    width: 3em;
    color: gray;
  }
  pre span.line-number:before {
    content: counter(line-number);
  }
</style>
