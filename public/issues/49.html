Feature request #49: REQUEST: iTunes volume control in menu bar (airport express remote volume)

Kind feature request
Product Synergy Advance
When Created 2004-11-30T21:43:55Z, updated 2006-07-05T02:03:27Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email: "Have you thought about putting a volume control next to the synergy buttons at top?"

A few people have requested this. iTunes volume control is already present in Synergy Classic in the form of hot keys, but I do plan to add a graphical control as well to Synergy Advance.
