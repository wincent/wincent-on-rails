Feature request #23: REQUEST: Use hot key to call up iTunes-style search field

Kind feature request
Product Synergy
When Created 2004-11-16T19:03:39Z, updated 2006-07-05T02:03:00Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email: "A quick way to get to the expression search feature where you can type in a partial song name, album name, etc and iTunes will filter the results for you in real time.

Perhaps you hit a hotkey, a small transparent window pops up like the current track indicator, you type in a string, hit enter, and it presents you with a list of valid results and then you arrow to the right one and hit enter again? Real time results filtering is probably unnecessary."

I already have a prototype of this functionality in Synergy Advance, so I am filing this feature request against Synergy.
