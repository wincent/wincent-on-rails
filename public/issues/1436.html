Feature request #1436: "Gists" or "pasties" like feature

Kind feature request
Product wincent.com
When Created 2009-11-06T16:07:56Z, updated 2010-09-09T10:08:38Z
Status closed
Reporter Greg Hurrell
Tags no tags

Description

I often want to write a slab of text that is simultaneously:

  • too long for a "tweet"
  • not on-topic enough for a blog post
  • too ephemeral to warrant a wiki article (ie something of only transitory importance, like some notes I'm making on a problem I'm working on)

So what's needed I think is another model in the application, akin to GitHub's "gists", or the ubiquitous "pasties".

Of course, I may be over-thinking this one and perhaps I should just be writing wiki articles for these things, perhaps tagging them appropriately with "gist" or "throwaway" or some other suitable tag.
