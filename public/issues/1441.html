  1. Greg Hurrell 2009-11-12T02:38:34Z

    Hi Abby,

    I edited out your email address and full name so that it wouldn't appear in public view.

    You could try doing a Spotlight search for "Synergy" to see where it is on your hard drive. The full name of the file your are looking for is either "Synergy Preferences.app" (for version 3.5 and above) or "Synergy.prefPane" (for versions older than 3.5).

    Once you have found and disposed of any old versions you might have on your hard drive, you can download Synergy again from:

    • https://wincent.com/products/synergy/download

    This will download a ZIP of the latest version (currently Synergy4.1.1.zip) to your default downloads folder (usually the "Downloads" folder inside your home folder).

    You unzip this by double-clicking on it. So that it doesn't go missing again, I recommend that you install it in an easy-to-find location such as your "Applications" folder. ("Installing" it really just means dragging and dropping it to your desired location.)

    Double-click the installed file to open the preferences window and set things up:

    • The "Start" button can be used to manually start Synergy
    • There is a checkbox to automatically launch Synergy whenever you login for convenience
    • If you need to re-enter your serial number you can do that in the "About" tab, where there is a button labelled "Enter serial number" (although it won't actually ask you for a serial number unless you deleted you existing preferences from the disk; these are stored at /Users/<your user name>/Library/Preferences/org.wincent.Synergy.plist)

    Let me know if you need any further help.
