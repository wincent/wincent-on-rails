Feature request #300: REQUEST: Hot key/menu item/button to disable Hot Keys

Kind feature request
Product Synergy
When Created 2005-09-16T19:26:31Z, updated 2006-07-05T02:06:27Z
Status open
Reporter Alexander Witkowski
Tags no tags

Description

Dear Wincent,

I would love to be able to disable the hot keys when using another app. that conflicts with the same keys, either by selecting something in the menu bar menu and maybe even hitting a hot key itself, without having to enter the System Preferences to disable them or turn off Synergy itself.

-Alex

Comments

  1. Greg Hurrell 2005-09-16T22:01:06Z

    Thanks for the request. See also request #11. I've reworded the summary for this request so that it doesn't overlap with #11. I think that there's room for both of these features in the app.

    Basically we have three possible ways to turn off hot keys:

    1. Pressing a hot key

    2. Choosing a menu item from the Global menu

    3. Hitting a button in the menu bar

    I'll probably implement 1 and 2, but I can't see 3 as being very popular so I doubt I'll go ahead with that. A fourth way would also be:

    4. Automatic disabling/enabling of hot keys based on frontmost application (see request #11).
