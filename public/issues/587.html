Feature request #587: REQUEST: Full-screen mode for Floater

Kind feature request
Product Synergy
When Created 2007-06-26T01:19:59Z, updated 2007-06-13T05:12:55Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

"A more interesting feature would be a Floater mode that filled the whole screen with a big image of the album cover and the track information in a large font, and that stayed on the screen like a screensaver or iTunes visualizer, that could be invoked with a keystroke and/or after a specified time of no user input while iTunes is playing.

The iTunes visualizer "Cover Version" does this already, except it can only be started from within iTunes, like any other visualizer, whereas my suggestion above would work no matter what app was in the foreground when the keystroke or idle-time activation occurred."

I think this is a good idea although I am not sure whether implementing a "full screen floater" is actually the best way to get what you want; there may be other, better solutions. For example, in Synergy Advance, there is already the ability to specify a global, system-wide hot key that allows you to activate an iTunes visualizer whenever you want; in other words, rather than implementing an independent full screen floater the best way to address your feature request might be to either add a hot key to Synergy that does the same thing and/or add the idle-time activation that you mention.

I'm putting this feature request open so that others can comment on what solution they'd prefer. I'll decide which way to go based on feedback.
