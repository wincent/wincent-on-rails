Feature request #1435: OpenID support

Kind feature request
Product wincent.com
When Created 2009-11-06T14:39:10Z, updated 2010-07-03T18:44:10Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

Really want two-way support here:

  • people should be able to log in using OpenID
  • the site should be able to serve as an OpenID server for people with user accounts here

The latter feature most likely won't be used by many people, but I certainly will use it (at the moment I'm running a third-party PHP script to do the same thing).
