Feature request #191: REQUEST: Option not to display anything is no album artwork is available

Kind feature request
Product Synergy
When Created 2005-04-18T02:29:33Z, updated 2006-07-05T02:05:09Z
Status open
Reporter NobodySpecial
Tags no tags

Description

I think it would be a good option to offer the ability not to display any image if there is no album artwork available. The current question mark ads a visual element that doesn't convey any more information than not showing anything at all.

Comments

  1. Greg Hurrell 2005-04-18T08:30:46Z

    Marking as ASSIGNED. If enough people vote for this feature I'll implement it. What the question mark is intended to indicate is, "You have your preferences set to display album art but none is available; if I find some on the Internet later on I will show it.", and it's intended to be visually different from the other preference option which is to show no art at all.
