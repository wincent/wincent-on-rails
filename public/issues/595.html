Bug #595: Buttons not working properly

Kind bug
Product Synergy
When Created 2007-07-28T14:34:17Z, updated 2007-07-29T06:14:59Z
Status open
Reporter Geoff
Tags no tags

Description

I have a problem that has not been remedied in versions 7.0.2 or 7.1 of iTunes.

The problem has not been solved by Synergy 3.1.4, as I thought might happen when I purchased this copy of Synergy.

With my earlier version of iTunes (version ????) I could listen to music as I worked on my eMac. The problem came about after I upgraded to 7.0.1. I used to be able to double-click on an item in a playlist and, when the piece finished, the next item in the playlist would start playing. This automatic playing continued until the last item in the playlist was finished.

Now I have no automatic continuation to the next item. If I want the next item in a playlist I have to double-click on it - very inconvenient to be stopping other work to play the next item.

I have the Synergy buttons at the top of my screen and have set hot-keys. When I click on the ‘next track’ button (or use the hot-key) the playback is like fast-forwarding through the current track and when the track is finished playing stops. If I want the next track I have to bring up iTunes and double-click as in the past with the iTunes updates.

Help!!!! (please).
