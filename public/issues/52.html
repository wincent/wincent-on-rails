Bug #52: Menu bar buttons don't respond to left-click events, but right-clicks work

Kind bug
Product Synergy
When Created 2004-12-03T07:10:28Z, updated 2006-07-05T02:03:27Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email: "I'm using synergy 1.4 and have one button (play/pause) in the menu bar, using the TigerWhite button set. After a while (possibly after waking from sleep?) it begins to not respond to clicking on it to pause or play, although right-clicking still yields the Synergy menu, and the keyboard controls still work. Changing the button set and then changing back gets it working again. I don't know if it's specific to this button set or not as it's the only one I like and have used."

This issue sounds suspiciously similar to bug #39 and bug #50, but filing it as a separate report for now.
