Feature request #112: REQUEST: Remote control over a network

Kind feature request
Product Synergy Advance
When Created 2005-02-03T10:34:03Z, updated 2008-11-11T12:51:34Z
Status open
Reporter Steven Skoczen
Tags no tags

Description

Hello,

This is a feature request for synergy to support remote-control of a networked iTunes via Remote Apple Events.

Functionality would be identical to the normal synergy, except that the commands would to to a specified networked iTunes.

There are quite a few applications that support this idea, but none with the polished interface or featureset of synergy. It would be good to have it all in one place!
