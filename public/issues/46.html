Kind feature request

Description

From a user email: "I use 2 monitors and itunes. When visualizer is active it blanks out the 2nd screen (no matter which has the itunes window)

Will your program stop that so i can work on one monitor and the other wilkl run visualizer."

This is something that only iTunes can really control, so you should submit a feature request to Apple:

http://www.apple.com/feedback/itunes.html
