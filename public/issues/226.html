Description

In the Synergy Global Menu, not all tracks by some artists are shown, even when selecting "All Albums". I haven't noticed any pattern in which don't show up, all I know is that some just don't.

Comments

  1. Greg Hurrell 2005-05-06T16:41:19Z

    Those tracks are missing because they have no album set.

    Thanks for reporting this. I did know about it but hadn't made a note of it here yet.

    It happens because Synergy Advance is basically asking, "What are all the albums for this artist? Show me the tracks from those albums..." and of course the tracks which don't have an album set for them don't get shown.

    I think I need to artificially make an "album" (a virtual album, not a real album) called "Unknown album" for this orphan tracks, then they should appear in those "All" submenus. I'll do my best to get this one fixed prior to the next preview release.

  2. Greg Hurrell 2005-05-16T03:12:48Z

    Ok, this is now fixed. Adding the fixed-in-prerelease keyword. You'll be able to see the fix when the 0.2 public preview release comes out at the end of this month.

    Basically, if an item has no Genre set it correctly winds up in the "All Genres" and "Unknown Genre" submenus. If an item has no Album set it correctly shows up in the "All Artists", "Untitled Album" and "All albums" submenus. And if neither Genre nor Album is set then it winds up in all of the above, as appropriate.
