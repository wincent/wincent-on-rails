Feature request #1485: Publish Synergy 4.4 via Amazon S3

Kind feature request
Product Synergy
When Created 2010-02-21T11:50:39Z, updated 2010-02-22T03:04:31Z
Status closed
Reporter Greg Hurrell
Tags no tags

Description

Just a reminder to myself: this is going to be the first release that I publish via Amazon S3.

Why? The data quantities involved are so small that S3 is effectively free, so we may as well use it to keep the application instance as lightly loaded and responsive as possible.

Comments

  1. Greg Hurrell 2010-02-21T12:09:15Z

    Quick reminder on how to do this:

    aws put s3.wincent.com/synergy/releases/synergy-4.4.zip \
        ~/local/path/to/Synergy4.4.zip

    Would upload Synergy4.4.zip to the bucket s3.wincent.com with key synergy/releases/synergy-4.4.zip, from where it could be downloaded using any of:

    • http://s3.wincent.com/synergy/releases/synergy-4.4.zip
    • http://s3.wincent.com.s3.amazonaws.com/synergy/releases/synergy-4.4.zip
    • http://s3.amazonaws.com/s3.wincent.com/synergy/releases/synergy-4.4.zip
  2. Greg Hurrell 2010-02-21T12:22:52Z

    And to be publicly readable:

    aws put s3.wincent.com/synergy/releases/synergy-4.4.zip?acl --set-acl=public-read

    Or more briefly:

    aws put s3.wincent.com/synergy/releases/synergy-4.4.zip?acl --public
  3. Greg Hurrell 2010-02-22T03:04:13Z

    It's out.
