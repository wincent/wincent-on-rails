Feature request #237: REQUEST: Submenus based on initial letter of artist names in Global Menu

Kind feature request

Status open
Reporter David Ameeti

Description

For those people who have a substantial number of different artists within iTunes, make the first pop-out menu for the artist list be the alphabet with a submenu listing each artist starting with that letter. This should create a submenu of 0-27 entries (depending on which letters actually have artists starting with that letter, as well as the catch-all of various/unknown artists).
