Feature request #325: REQUEST: Add Hot Keys for Fast Forward / Reverse

Kind feature request
Product Synergy
When 2005-10-30T12:45:18Z
Status closed
Reporter Synergy User
Tags no tags

Description

In iTunes itself, there are hot keys for Fast Forward / reverse [CMD]-[OPT]-[RIGHT ARROW] and [CMD]-[OPT]-[LEFT ARROW]. Unfortunately, you must have iTunes as the frontmost application to use them.

REQUEST: Add Hot Keys to allow this while iTunes is int eh Background.

This is desirable because on long Podcasts there are many places where you want to skip over ads or dead spots - without having to switch back into iTunes, and then hit the Hot Key, then switch back to your working application.

Also, a ReplayTV or TIVO Like function that would be nice is a quick skip forward 30 Seconds Hot Key.

Comments

  1. Greg Hurrell 2005-10-30T19:29:21Z

    If you press-and-hold (rather than press-and-release) the "Next" and "Previous" hot keys then you'll get the desired effect (fast forwarding and rewinding within a track rather than skipping whole tracks). This should work for any track type that supports fast forwarding or rewinding.

    For reference, Synergy Advance works in the same way but also adds a few extra hot keys, like optional separate keys for fast forwarding and rewinding, keys for skipping a fixed number of seconds forward and back etc.
