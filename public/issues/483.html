Feature request #483: REQUEST: "Recent albums" in the menu (not just recent tracks)

Kind feature request
Product Synergy
When Created 2006-10-05T05:24:20Z, updated 2007-01-12T10:53:47Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

I'd like to have a way of showing recent Albums in the Menu. I'm kind of "old school" - I listen to Albums, one a time. It would be great to show the top ten most recent Albums I've listened to so that I can just click on an Album I listened to yesterday and Synergy would just play it.

Thanks for the feature request. I'd just need to figure out the neatest way to implement this in the UI. Either as an optional "Recent Albums" submenu (contrast this with a "Recent Tracks" submenu); or putting the albums (again, optionally) "in line" in the menu like this:

+ Recent Tracks + Recent Track 1 + Recent Track 2 + Recent Track 3 + + Recent Albums + Recent Album 1 + Recent Album 2 + Recent Album 3 +
