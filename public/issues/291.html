Feature request #291: REQUEST: Option to show "Grouping" in Floater

Kind feature request
Product Synergy
When Created 2005-09-01T08:49:06Z, updated 2007-06-11T10:43:35Z
Status closed
Reporter Guy
Tags no tags

Description

Currently, there is no option to show the "Grouping" of which the track belongs to. This is especially needed when listening to classical music where many individual tracks have ambiguous names, such as "Allegro" or "Presto", without any indication of which specific symphony or concerto they belong to.

My suggestion is to optionally add a second line of smaller text just beneath the track name (but keeping it above the divider line)

Thank you
