Feature request #1536: Issue stats (new, open, closed counts; per-product, per-kind counts etc)

Kind feature request
Product wincent.com
When 2010-04-14T17:43:23Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

Could possibly be interesting. Not necessarily very useful, but I certainly do think it could be interesting.

Right now I find myself showing all issues for a product and then sorting by status. It might be quicker to just look at a table showing all products and then clickable counts which take me to a specific page showing only the relevant issues.
