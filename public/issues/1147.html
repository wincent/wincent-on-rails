Bug #1147: iPhone' songs are played (instead of iTunes library) when Syngergy ask iTunes to play a song

Kind bug
Product Synergy
When Created 2008-10-05T08:48:13Z, updated 2008-10-06T11:42:18Z
Status open
Reporter anonymous
Tags no tags

Description

When iTunes isn't started, if I type the "play" shortcut of Synergy, it launchs iTunes and starts to play a song in my iTunes library.

If iTunes is started and playing no music, if I type the "play" shortcut, iTunes plays a track from my library.

I noticed a strange behaviour when I connect my iPhone to the mac, if iTunes isn't playing a file and I type the "play" shortcut of Synergy, iTunes start to play a song from my iPhone, not on my Library.
