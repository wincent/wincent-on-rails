Feature request #126: REQUEST: Add a Rating Menu to the Global Menu

Kind feature request
Product Synergy Advance
When Created 2005-03-06T13:45:50Z, updated 2005-05-04T02:47:45Z
Status closed
Reporter Britney
Tags no tags

Description

I would like to be able to rate the item that is currently playing from the Global Menu. This could be implemented using a Rating submenu or better yet something like the Finder's Color Label contextual menu using stars instead of the colored circles.
