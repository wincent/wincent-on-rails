Feature request #395: REQUEST: Show floater in front of full-screen videos

Kind feature request
Product Synergy
When Created 2006-04-04T10:04:31Z, updated 2006-07-05T02:07:36Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From this forum thread:

"I have been using iTunes as a music video jukebox player on an intel mini hooked up to my living room TV. Everything works great, except there's no indication of what video is playing, so you have to un-full screen, and read the itunes indicator.

I had the brilliant idea today to install Synergy (which I own and use on my other mac) to make "MTV-Like" titles in the lower-right corner of the screen, thus telling me what artist + song is playing.

Sadly for me, it seems synergy displays its track info underneath the full screen video.

I realize (broadly) the challenges involved in fixing this, but I thought I'd throw my 2 cents in for it as a feature request. If you could get this to work, it would add a much-needed benefit to full-screen video viewing. "

I am not sure if this is possible, but will look into it. Synergy tries to put its floater on top of everything, but not so far to the front that it would appear in front of the screensaver, for example. I am not sure at what level iTunes shows full-screen videos, or whether it is possible to get in front of that level. The same really applies when watching iTunes visualizers in full-screen mode.

See also feature request #343 (ironically, requesting that the floater be *suppressed* when playing videos). Basically these two feature requests are mutually exclusive so they'd both need to be optional in the preferences.
