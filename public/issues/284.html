Bug #284: Screen Redraw Bug

Kind bug
Product Synergy Advance
When 2005-08-02T03:36:07Z
Status closed
Reporter Jason Snell
Tags no tags

Description

Synergy Advance causes an odd (almost imperceptible) redraw bug in a square portion of my screen. I can see it when I turn on the update-highlight feature of Quartz Debug, and it shows that square being highlighted constantly, as if it's endlessly being redrawn. In normal use, a square of my screen updates noticeably later than the rest of the screen; it's most noticeable (as you might expect) during drags. And it stops as soon as I quit Synergy Advance, although it doesn't start up again when I re-launch it. I use a PowerBook and switch between external-monitor and internal-monitor configurations every day; it's possible that this odd screen effect occurs only when a display switch has occurred.

Screen shot at http://www.intertext.com/quartzdebug.jpg

Quitting Growl had no effect on this bug; only quitting Synergy Advance.
