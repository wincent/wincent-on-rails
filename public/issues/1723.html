Kind bug
Product Command-T
When 2010-11-03T15:22:05Z

Tags no tags

Description

On newer versions of MacVim, the CommandT prompt is not visible most of the time. When typing text, the prompt will appear and disappear as keys are pressed.

This does not happen on older versions of MacVim. In-particular, MacVim 7.2 stable 1.2 (33.3) does not exhibit this bug.
