Feature request #1474: Use tool-tips to show just-in-time information about form fields

Kind feature request
Product wincent.com
When 2010-02-07T08:38:08Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

As an example, the form that I'm writing in right now to create this ticket has a number of labels with additional information to guide completion:

  • Summary: "a one-line summary of your issue"
  • Product: "optional (leave blank for general issues)"
  • Description: "please be as detailed as possible (wikitext markup help)"

Not all of these fields are really amenable to tool-tips (the pop-up menu, for example) but others might be.

For an example of what these tooltips might look like in practice see this page.
