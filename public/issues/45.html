Bug #45: Synergy does not recognise when iTunes launches or exits

Kind bug
Product Synergy
When Created 2004-11-27T01:46:58Z, updated 2010-08-12T09:48:05Z
Status closed
Reporter Cody Walton
Tags no tags

Description

The previous version (1.4 i believe) worked just fine. After upgrading to 1.5b, synergy no longer displays any text in the floater, only the cover art. Then after starting and restarting Synergy in the preference pane, it no longer recognizes when iTunes starts or stops. I tried moving the preference file, changing the floater size/positon and restarting, uninstalling and reinstalling, but nothing worked.I am running Mac OS X 10.3.6 on a 1.8 GHz G5 with 512 ram. I do have an external GVP firewire drive, and I use Shapeshifter 2.0 and Konfabulator, but these have never had a conflict with Synergy before. My exported system profile can be found here: http://homepage.mac.com/codywalton/.Documents/ JCW-System-Profile.txt
