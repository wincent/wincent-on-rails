Feature request #435: REQUEST: Option-Click Next and Previous should choose Next/Prev. Album (as it does in iTunes)

Kind feature request
Product Synergy
When Created 2006-06-28T19:01:37Z, updated 2007-08-26T05:19:35Z
Status open

Description

Title pretty much says it all - and Option-Click on the track skip buttons in iTunes changes to the next or previous album. Synergy should replicate this behavior. I just purchased before checking this and was a bit disappointed that it wasn't included. Great app otherwise! Thanks!
