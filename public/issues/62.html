Feature request #62: REQUEST: Randomize button style when iTunes launches, or after an interval

Kind feature request
Product Synergy
When Created 2004-12-09T02:39:01Z, updated 2006-07-05T02:03:34Z
Status open
Reporter sjmills
Tags no tags

Description

Since my computer rarely ever has to restart, the "Choose random button style when Synergy launches" pref rarely has a chance to do anything. But since I launch iTunes almost every day, it might be more fun/handy/whatever to also have a "Choose random button style when iTunes launches" pref.
