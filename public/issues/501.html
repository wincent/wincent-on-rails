Feature request #501: REQUEST: Link to Audioscrobbler profile, statistics etc

Kind feature request
Product Synergy
When Created 2006-11-14T03:49:06Z, updated 2006-12-05T00:41:27Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

As indicated in the screenshot of the original Audioscrobbler feature request (feature request #108), it would be nice to have some menu items for quickly jumping to certain Audioscrobbler-related pages.

The screenshot itself is attachment #12. It shows:

- a link to the Audioscrobbler home page - a link to the user's personal statistics page - a "Top lists (local)" link

I personally would be inclined to stick all Audioscrobbler-related items in a submenu; that way new items can be added without worrying about cluttering up the main menu. I don't think I would add a local version of the top lists; seems like re-inventing the wheel when I could just link to the corresponding page on Audioscrobbler.
