Feature request #1416: use floater for pause/play information

Kind feature request
Product Synergy
When 2009-09-29T10:50:11Z
Status open
Reporter welchb
Tags no tags

Description

Hey man,

I recently purchased synergy after trying it out for a while with the Machiest - Love it.

But it doesn't quiiite do what I would really like it to do. I'd like it to use the floater to pop up when you have the "extra visual feedback on play/pause", not growl. The floater is larger with more information and I'd like to use that. Is there any way you could provide a way to select *which* notification method you use for this option?

Reasoning: I don't mind growl I use it in other places. But it's sort of sadly lacking in size/information for iTunes/synergy's needs. And since you can't just use growl for this "extra" thing (it's all or nothin') that means you get two notifications for each new track, which is sort of goofy.

Thanks much, man!

-bw

Comments

  1. Greg Hurrell 2009-09-29T13:46:43Z

    Perhaps the "extra visual feedback" option should be worded a little more clearly, but it really was only ever intended to apply to the centered "bezels" that you see when you press hot key combinations. I didn't use the word "bezels" because that's geek/programmer terminology for them.

    The fact that Growl notifications get shown on play/pause is actually just a coincidence stemming from the way the Growl system works: basically, iTunes sends a "notification" for each state change (from playing to paused, or vice versa) and Synergy just relays that to Growl.

    Synergy's own built-in floater has never behaved that way and was always intended to work like the informative overlay that you see when a music video starts playing. That's why it has never been shown when resuming paused playback, and has only been shown when a new track starts playing from the beginning.

    Sounds like what you really want isn't actually a modification of the "Extra visual feedback" checkbox, nor really a way to control which notification method gets used for a given thing. I think what you actually want is a new checkbox in the Advanced tab called something like "Show Floater when resuming playback", with a tooltip that explains something like "Normally, the Floater is only displayed when a new track starts playing for the first time; this setting can be used to also show it when resuming paused playback".

    Not sure if that would be clear enough. What do you think?
