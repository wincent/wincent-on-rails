Feature request #583: REQUEST: Auto-pause when screensaver activates

Kind feature request
Product Synergy Advance
When Created 2007-06-11T09:03:20Z, updated 2007-10-01T10:51:16Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

Splitting this off from feature request #217, where this appeared as a comment:

I'd definitely like to see an option to automatically pause iTunes when the screen saver is activated. I have a hot corner set up to quickly lock my screen, but if I'm listening to music and forget to pause iTunes I have to unlock the screen again just so I can pause it.

Comments

  1. Greg Hurrell 2007-06-11T09:06:42Z

    I think the automatic aspect of it is worth implementing, but there is one thing I wanted to respond to in your comment:

    The hot keys will continue to work even when the screensaver is active, so you do already have a means of pausing your music without unlocking the screen. Note that if the password dialog is showing you'll have to dimiss it first with the Escape key (or just wait for it to go away) because otherwise it will capture the keyboard and prevent the hot keys from getting through.

  2. ckd 2007-10-01T10:51:16Z

    The hot keys used to work, but there was a security fix some time ago which prevented keystrokes from being passed through the screen saver and they no longer work even if you dismiss the password dialog and hit the key repeatedly.
