Bug #580: Synergy doesn't display MPEG stream metadata

Kind bug
Product Synergy

Status open

Description

Playing an MPEG stream in iTunes which also caries metadata for each song causes the Growl floater to be displayed with the name of the stream only and a random album picture. The floater is displayed each time a new song starts playing in the stream. Metadata being displayed in iTunes, it would be nice if Synergy could also announces songs playing.
