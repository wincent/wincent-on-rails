Bug #449: Volume hot keys stop responding after running for a few days

Kind bug
Product Synergy Advance
When Created 2006-07-22T13:49:41Z, updated 2009-09-03T15:40:13Z
Status open
Reporter john
Tags no tags

Description

After a few days of running Synergy Advance will suddenly stop responding to my increase and decrease volume hot keys, which I have set to ctrl-option-up arrow and ctrl-option-down arrow respectively. The only way to get them to work again is to quit Synergy Advance and re-launch it; previously I tried just opening the preferences, removing the hot keys and re-adding them but they still did not work.

I wasn't sure where to find any useful debug info for when this is happening so if you could point me to where to look I will gather that information the next time the problem arises and attach it here.
