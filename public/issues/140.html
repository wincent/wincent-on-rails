Bug #140: Show Synergy artwork in iTunes too

Kind bug
Product Synergy
When 2005-03-13T00:13:55Z
Status closed
Reporter Eerk
Tags no tags

Description

Hello Wincent,

While working with your fixed 1.7.1 version I had iTunes and the floater open at the same time to check the response from keystrokes. At first it stroke me as odd that some artwork was visible in both iTunes and the floater and other artwork only displayed in the floater. I quickly realised that this is the artwork Synergy downloads separately for itself.

The proposed enhancement: is it possible to implement an option to have Synergy 'upload' the artwork to iTunes, preferably for the whole album. Since this enlarges the iTunes files perhaps a 'on request only' preference is in order too.

What do you think?

Regards,

Eerk.
