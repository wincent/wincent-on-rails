Bug #703: Synergy must be forced to start, doesn't start when system and then iTunes is started

Kind bug

Status closed

Tags no tags

Description

At system startup and launching iTunes, Synergy doesn't start. Have to go into system preferences and manually start Synergy.

Comments

  1. Greg Hurrell 2008-05-07T20:51:07Z

    Perhaps your login items are not up to date. Try this:

    1. In the Synergy preferences, uncheck the "Launch at login" checkbox
    2. Click "Apply"
    3. Now re-check the checkbox
    4. Click "Apply"

    That should fix up any stale entries in the login items.

    As for starting when iTunes starts, that would require a third process to watch for iTunes being launched, which would then launch Synergy. Rather than adding a third process into the mix it is probably more efficient to just run Synergy at login anyway.
