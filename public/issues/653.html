  1. Greg Hurrell 2008-02-23T10:33:04Z

    Are you using 3.5a5? The failure to launch at login should have been fixed as reported in bug #605:

    http://wincent.com/a/support/bugs/show_bug.cgi?id=605

    It may just be that you have a stale entry in your login items. To fix that you would force a refresh as follows:

    1. Uncheck the "Launch at login" checkbox 2. Click "Apply" 3. Check the checkbox 4. Click "Apply"

    I am going to close this as a duplicate. We'll re-open #605 if necessary.

    As for the hangs, I'll need more information, which would need to go in a separate report anyway. If you're using 3.5a5 you might be seeing this issue:

    http://wincent.com/a/support/bugs/show_bug.cgi?id=640
