Feature request #1475: Use tool-tips to provide alternative download options

Kind feature request
Product wincent.com
When Created 2010-02-07T08:42:28Z, updated 2010-04-14T17:46:37Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

Currently on the Synergy product pages the download button takes you to a separate product page which offers different download options. The default option is automatically downloaded after a delay.

It would be nicer to have the following behaviour:

  • Hovering over the download button would reveal all download options
  • Clicking on the download button would download the default option
  • Possibly, as a fallback: in browsers without JavaScript clicking on the download button would take people to the existing download page which shows the different options

For an example of what this might look like see this page.
