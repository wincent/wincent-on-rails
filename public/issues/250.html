Feature request #250: REQUEST: Show Internet radio tracks in the "Recent tracks" listing

Kind feature request
Product Synergy
When Created 2005-05-26T18:50:10Z, updated 2006-07-05T02:05:48Z
Status open
Reporter Greg Hurrell
Tags no tags

Description

From a user email:

"Be nice if the 'recent tracks' drop down showed the artist and song for streaming radio stations. These details show up in the floater so the info is getting to Synergy but not being logged."

I think this would be good as an option. Of course, you can't actually skip back and forth between tracks on an Internet radio station so perhaps the items should be ghosted. There are a few options:

1. Show one entry per radio station (no song information, only show the name of the station); these items would work and you'd be able to select them to switch between radio stations.

2. Show one entry per song (unghosted); selecting previous tracks from the same radio station would have no effect, selecting ones from a different station would switch to that station.

3. Show one entry per radio station (selectable), and list specific tracks underneath that entry (indented, ghosted).

4. Similar to "3", but show the tracks in a submenu attached to the radio station.

Out of these I probably prefer "3" or "4" the most.

See also bug #163.
