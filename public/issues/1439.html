Kind feature request
Product wincent.com
When 2009-11-11T15:14:38Z
Status open
Reporter Greg Hurrell

Description

It is probably a good idea to use Screw.Unit to test as much of the JavaScript as can be tested that way, thus reducing dependence on Selenium/Cucumber.
