Feature request #260: REQUEST: Adium support

Kind feature request
Product Synergy Advance
When Created 2005-06-16T17:09:00Z, updated 2006-07-05T02:06:06Z
Status open
Reporter William Styler
Tags no tags

Description

Hi there. This may be a bit of a stretch, but I mostly use Adium (http://www.adiumx.com/) for Instant Messaging, and I was wondering if there would be any way that even a few of the "Show current song" and/or "Update Icon with album art" features available with iChat could be made to work with Adium as well. With regards to implementation, I believe Adium does support Applescript for dealing with matters of status.

This is by no means critical, I just thought I'd put it out there. Thanks again for the great product!

Will

  1. Greg Hurrell 2005-06-16T21:20:16Z

    Thanks for filing the feature request, Will. My plan all along has been to add support for other IM clients. I am not sure what level of AppleScript support Adium has, but Synergy Advance is designed with an abstraction layer so that it can work with multiple IM clients. Even the communication with iTunes occurs via an abstraction layer so that I will be able to add support for other audio players in the future. The only limitation is imposed by the level of AppleScript support that a given application has; if it has no or limited supported then Synergy Advance won't be able to do much with it.

    It's just a matter of writing the plug-ins for these other apps. I am going to add 1.0 as the "milestone" because before tackling that job I want to get the rest of the functionality in place (floater, menu bar buttons, cover downloads etc) to bring Synergy Advance up to par with Synergy. Then I'll get onto adding this extra stuff.
