The installer seemed to run OK to completion, but I cannot find the executable or man page anywhere. I even tried the "locate" shell command:

  1. Greg Hurrell 2011-04-27T23:58:20Z

    The locate database is only periodically updated (once a day, if I recall correctly), so you're unlikely to see results for stuff that you just installed.

    For something that is always up-to-date, you could use mdfind (command line front-end to Spotlight).

    In any case, the installer is supposed to put the tool inside /usr/local/bin/ (again, working from memory here), so have a look there. Perhaps /usr/local/bin is not on your PATH.
